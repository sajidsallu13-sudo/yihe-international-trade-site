YIHE INTERNATIONAL TRADE SERVICES — website scaffold

What's included:
- Next.js frontend with i18n (en/zh)
- Simple blog pages
- Supplier registration and contact forms
- Express backend (server/index.js) with Postgres-ready endpoints

Quick start:
1. Install dependencies: npm install
2. Set environment variables (see .env.example)
3. Run locally: npm run dev

Deployment suggestions:
- Frontend: Vercel
- Backend / DB: Railway, Render, or PlanetScale (Postgres)
